# Beyblade X

Questo repository raccoglie un piccolo progetto ludico dedicato alla collezione di Beyblade X di famiglia.

L'idea e' creare uno strumento semplice e divertente per far giocare i miei figli con i loro Beyblade: esplorare la collezione, confrontare i pezzi, comporre combinazioni e scegliere in modo casuale nuovi abbinamenti da provare durante le sfide.

## Cosa contiene

- `progetto/`: sito statico in HTML, CSS e JavaScript per visualizzare e usare la collezione.
- `progetto/assets/`: dati, immagini e script usati dall'applicazione.
- `json_schema.json`: schema di riferimento per descrivere i dati dei Beyblade.
- `esempio_beyblade.json`: esempio di struttura dati per un singolo Beyblade.
- `punte_reference.json`: dati di riferimento per le punte.
- `beyblades/`: materiale relativo ai Beyblade della collezione.
- `foto retro/`: fotografie o immagini di supporto.

## Scopo

Il progetto non nasce come applicazione professionale, ma come gioco di famiglia: un modo per trasformare una collezione fisica in una piccola esperienza digitale, utile per inventare sfide, scegliere combinazioni e rendere piu' varia la fase di gioco.

Per i dettagli tecnici su come aprire il sito e su come sono organizzati i dati, vedere il README dentro la cartella `progetto/`.
